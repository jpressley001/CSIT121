JavaScript and XML

What is JavaScript?

JavaScript creates interactive effects on the web pages by "manipulating objects (broswer windows,documents, and elements such as hyperlinks,forms,and images) associated with the web pages documents" (pg.279). It is an object-based, and client-scripting language.

How is JavaScript related to HTML and CSS?

JavaScript statements are coded with html tags or html script elements by directly being coded on web pages. HTML creates webpages structures (headers, headings,paragraphs, etc), CSS styles and format webpages (borders,color,font-family, background and more), and JavaScript adss dynamic effects to the webpages such as responding to the mouse movements or the loading of the pages, etc.

What are some elements of JavaScript?

JavaScript uses variable, loops, functions and conditional statements. The length prperty returns the number of elements in a array. The push() method adds elements to the end of an array. ParseFloat() converts a string to an interger.

What is XML?

eXtensible Markup Language (XML), "text-based syntax that's designed to describe, deliver and exchange structured information" (pg.22). It was made to "extend HTML by separating data from presentation"(pg.22). It used flexible methods to HTML.

How is XML used?

XMl is used to store and transport data. It uses tags to define the attributes and elements of elements used.
